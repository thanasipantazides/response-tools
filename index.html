The FOXSI Response Tools documentation! 🦊

Welcome! This is the online documentation for the response-tools code repository developed for the FOXSI-4 sounding rocket mission.

The repository contains a Python package that can help a user download FOXSI’s telescope response elements. Additionally, a codebase exists to load in and work with the response files.

If you’d like to work with photon event lists from the FOXSI-4 flight, please see the foxsi4-science-tools repository instead.

Documentation/help can be found in the Response Tools Data Guide, the top-level Github repository README, and in the online documentation.
